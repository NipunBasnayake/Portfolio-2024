Todo List

ToDo List is a simple and interactive web application built using HTML, CSS, and JavaScript. It enables users to add, edit, and delete tasks, keeping their daily activities organized. The app features a clean and responsive design, with smooth animations and real-time updates. Perfect for managing personal tasks efficiently, it also includes features like task prioritization and completion status tracking.

Fashion Shop

FashionShop is a sleek and intuitive Java Swing-based desktop application designed to streamline fashion retail operations. This app enables users to manage inventory, track sales, and handle customer data with ease. Built with a modern and user-friendly interface, it incorporates features such as dynamic data display, sorting, and customized reporting, making it a robust tool for small to medium-sized fashion businesses.

Service Management System is a dynamic web-based solution developed using PHP, HTML, and CSS to manage service-based businesses. The system allows users to handle bookings, manage customer records, track services, and generate reports. Designed with a responsive UI, it ensures seamless accessibility across devices, offering a reliable tool for service providers to enhance their efficiency.
